JavaScript 是一门脚本语言，不是真正的编程语言，必须运行与浏览器中
数据类型：
        一、基本数据类型
        1、number，string，boolean，undefined   
        2、引用类型
运算：
    非NaN、非0、非undefined == TRUE == 1；
    
if 会发生隐式类型转换；
switch 不会发生隐式类型转换

函数中没有加return 自动返回undefined，相当于没有返回值
            加return，返回确切的值
 等号的右边就是表达式
            var a = 5 + 6;
等号的右边的是函数==函数表达式；


函数声明方式，可以先声明在调用，也可以先调用在声明
    是因为：函数提升---》js解析先把所有function解析提到前面去，表面上是先调用在声明，实际是先声明在调用。
函数表达式方式，只能先声明在调用。
